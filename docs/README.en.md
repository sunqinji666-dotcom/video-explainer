# Video Explainer

A multimodal video analysis workspace for turning footage into structured explanations.

![video-explainer project visual](assets/video-explainer-hero.png)

## What it does

A multimodal video analysis workspace for turning footage into structured explanations. It is designed to be understandable, local-first where applicable, and configurable without committing credentials.

## Quick start

Read the project README for the platform-specific command. Keep secrets in environment variables or local configuration files that are ignored by Git.

## Privacy boundary

The public repository contains no personal captures, credentials, tokens, or deployment secrets. Supply your own data locally.
