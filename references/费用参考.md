# 费用参考

基于 qwen3.7-plus 阿里云百炼官方定价（2026年6月查询）：

## Token单价

| 计费项 | 价格 |
|--------|------|
| 输入（0<Token≤256K） | 2元/百万Token = 0.002元/千Token |
| 输出（0<Token≤256K） | 8元/百万Token = 0.008元/千Token |
| 输入（256K<Token≤1M） | 6元/百万Token |
| 输出（256K<Token≤1M） | 24元/百万Token |

## 典型费用速查

| 5秒 | ¥0.02-0.03 |

| 1.5分钟（6段分析） | ¥0.40-0.80 |

## 费用说明

- 实际费用取决于视频内容复杂度（帧数、分辨率影响视频Token数）
- OSS上传费几乎可忽略（~¥0.001/次）
- 阿里云Paraformer转写：0.00008元/秒（约¥0.01/分钟）
- 每次分析前会显示预估费用并等待确认
